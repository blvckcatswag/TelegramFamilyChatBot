# Family Chat Bot

Телеграм-бот для семейного чата. Игры, напоминалки, погода, цитаты, дуэли и награды - всё через кнопки, чтобы даже мама разобралась без инструкций.

---

## Зачем это

В каждой семье есть чат в телеграме. Обычно там «купи хлеб», «ок» и тишина. Этот бот делает чат местом, куда хочется заглянуть - полить кактус, покормить кота, вызвать брата на дуэль или узнать, кто чаще всех говорит «ясно».

## Что умеет

### Игры

- **Кактус** - поливаешь раз в день, он растёт на 1 см. Но есть 1% шанс уколоться и уйти в мут
- **Кот** - кормишь раз в день. Может быть благодарным и прибраться (+порядок), а может разнести кухню (-порядок)
- **Порядок дома** - общий счётчик от 0 до 100%. Зависит от кота. В чате показывается прогресс-баром
- **Дуэль** - вызываешь соперника, 50 на 50 - проигравший в муте
- **Рулетка** - барабан на 6 позиций, патрон один. Кто-то точно замолчит
- **Пасхалка** - с шансом 0.01% кот опрокидывает кактус. Штрафы всем, хаос

### Напоминания

Создаёшь через диалог с кнопками - одноразовые или повторяющиеся. Бот их хранит в базе, так что после перезапуска ничего не теряется.

### Погода

Каждое утро присылает погоду по твоим городам. Можно добавить до 5 штук на чат. Если API прилёг - бот не падает, просто скажет что данных нет.

### Цитаты

Ответь на сообщение командой `/quote` - и оно сохранено. Потом можно вытащить случайную цитату, посмотреть последние или узнать, кого цитируют чаще всех. Каждое утро бот сам кидает цитату дня.

### Дни рождения

Владелец чата добавляет дни рождения через `/birthday_add Мама 25.03`. В нужный день бот поздравит, а за день до этого напомнит.

### Переводчик «ясно»

Бот ловит слова типа «ясно», «ок», «угу», «понятно» - и с шансом 35% отвечает шуткой. Есть счётчик, кто больше всех попался.

### Реакции

Бот следит за реакциями на сообщения. Можно посмотреть топ-5 эмодзи за месяц и кто собирает больше всех реакций.

### Ежемесячные награды

В последний день месяца бот автоматически подводит итоги и раздаёт звания:

- **Колючий садовод** - самый высокий кактус
- **Любимец кота** - лучший результат с котом
- **Дуэлянт** - больше всех побед в дуэлях
- **Везунчик** - чаще всех выживал в рулетке
- **Клоун месяца** - собрал больше всего клоунских реакций
- **Любимчик** - больше всего реакций в целом
- **Цитата месяца** - чаще всех цитировали
- **Мастер ясно** - чемпион по «ясно»

### Настройки

Владелец чата включает и выключает модули кнопками. Кто хочет - может пользоваться командами, но кнопок хватает.

### Публичный режим

Бот работает в любом чате. Добавляешь - он сам регистрируется, здоровается и назначает тебя владельцем. Никакой ручной настройки.

## Структура проекта

```
family-chat-bot/
├── main.py                  # точка входа
├── app/
│   ├── bot/                 # хэндлеры, клавиатуры, мидлвари
│   ├── config/              # настройки из .env
│   ├── db/                  # SQLite - схема и репозитории
│   ├── scheduler/           # APScheduler - крон-задачи
│   ├── services/
│   │   ├── admin/           # команды суперадмина
│   │   ├── awards/          # ежемесячные награды
│   │   ├── birthdays/       # дни рождения
│   │   ├── games/           # кактус, кот, дуэль, рулетка
│   │   ├── quotes/          # цитаты
│   │   ├── reactions/       # отслеживание реакций
│   │   ├── reminders/       # напоминания
│   │   ├── translator/      # переводчик «ясно»
│   │   └── weather/         # погода
│   └── utils/               # вспомогательные функции
├── tests/                   # 28 тестов
├── requirements.txt
└── .env.example
```

## Стек

- **Python 3.11+** - лучшая экосистема для телеграм-ботов
- **aiogram 3.x** - асинхронный, с роутерами и FSM, стандарт для новых проектов
- **SQLite + aiosqlite** - файл на диске, ничего настраивать не надо, потом можно мигрировать на PostgreSQL одной строкой
- **APScheduler** - крон-задачи с хранением в той же SQLite, напоминания переживают перезапуск
- **OpenWeatherMap** - 1000 запросов в день бесплатно, нам хватает с запасом

Без лишнего. Без Redis там, где хватает SQLite. Без Celery там, где справляется APScheduler.

## Как запустить

```bash
# клонируем
git clone https://github.com/blvckcatswag/TelegramFamilyChatBot.git
cd TelegramFamilyChatBot

# виртуальное окружение
python -m venv venv
source venv/bin/activate  # linux/mac
venv\Scripts\activate     # windows

# зависимости
pip install -r requirements.txt

# настройка - заполнить токены
cp .env.example .env

# поехали
python main.py
```

## Роли

- **SUPERADMIN** - разработчик, user_id в .env. Видит все чаты, может банить и делать рассылку
- **OWNER** - кто добавил бота в чат. Управляет настройками, городами погоды, днями рождения
- **USER** - все остальные. Играют, сохраняют цитаты, ставят напоминания

## Деплой

**Railway** - самый простой вариант. Подключил гитхаб, нажал деплой, бот работает.

**VPS Ubuntu** - для продакшна:
```bash
git clone {repo} && cd TelegramFamilyChatBot
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env && nano .env
sudo systemctl enable --now family-bot
```

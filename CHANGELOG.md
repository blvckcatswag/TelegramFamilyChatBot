## [2026-03-07] — Фидбек 2.0, Донат, Фиксы навигации

### Добавлено
- Категории фидбека: 🐛 Баг, 💡 Идея, 😤 Жалоба — выбор перед отправкой
- Таблица `Feedback` в БД с сохранением всех обращений
- Команда `/backlog` (только суперадмин) — список открытых обращений с пагинацией и кнопкой закрыть
- Фидбек принимает не только текст но и фото, голосовые, кружочки, видео, стикеры
- Ответ суперадмина через reply на карточку фидбека — доставляется пользователю в лс или в чат
- Донат через Telegram Stars: кнопки 5/10/25/50/100 ⭐ и свой вариант
- Донат крипто: адреса USDT TRC20 и TON (настраиваются через .env)
- Кнопка "💝 Поддержать" в главном меню, команда `/donate`
- Уведомление суперадмину при успешной оплате звёздами
- Автомиграция чата при апгрейде группы до супергруппы (TelegramMigrateToChat)
- Уведомление суперадмину об успешной или неудачной миграции чата

### Исправлено
- Кнопки reply-клавиатуры во время FSM-флоу (напоминания, фидбек, донат, дни рождения) теперь сбрасывают текущий флоу и выполняют навигацию, а не бросают ошибку

## [2026-03-06] — Reply-клавиатура, Дни рождения, Фиксы

### Добавлено
- Полноценная reply-клавиатура с многоуровневой навигацией (старт → меню → подменю)
- Подменю: Игры, Напоминания, Погода, Цитаты, Статистика, Справка, Настройки
- Все игры и функции доступны через кнопки без команд
- Категории цитат в меню (Золотой фонд, Тёмная лошадка и др.)
- Меню дней рождения с FSM-флоу добавления и удаления
- Команда `/cancel` для выхода из любого FSM

### Исправлено
- Команды слэш скрыты в группах (кнопка "/" не появляется в инпуте)
- Сообщения с кнопками удаляются после нажатия чтобы не засорять чат
- Краш в process_reminder_time когда message.text равно None
- TelegramBadRequest при повторном открытии списка напоминаний
- Застрявший FSM при создании напоминания
- Глобальная защита от TelegramBadRequest через safe_edit_text/safe_edit_reply_markup

## [2026-03-05] — Логирование, Sentry, Реакции, Награды

### Добавлено
- Централизованное логирование через logging_config.py
- Интеграция с Sentry (ошибки в прод)
- Глобальный error handler с уведомлением суперадмина в Telegram
- Таблица MessageAuthor для отслеживания авторов сообщений
- Команды /my_reactions и /top_reactions
- Ежемесячные награды: Клоун (больше всего 🤡) и Любимчик (больше всего реакций)
- README обновлён: PostgreSQL, Sentry, таблица переменных окружения

### Исправлено
- to_user_id = NULL в реакциях из-за отсутствия MessageAuthor
- Скрытие токена бота из URL в Sentry
- TelegramBadRequest при повторном нажатии на кнопки цитат

## [2026-03-02] — MVP и первые итерации

### Добавлено
- MVP: кактус, кот, порядок дома, дуэль, рулетка
- Напоминания с FSM (once/daily/weekly)
- Погода через OpenWeatherMap, рассылка по расписанию
- Цитатник (/quote, /quote_random, /quote_last)
- Переводчик "ясно" с вероятностью
- Роли: SUPERADMIN, OWNER, USER
- Поддержка PostgreSQL (prod) и SQLite (dev/tests)
- Деплой на Railway
- APScheduler с SQLAlchemy jobstore для персистентности заданий
- 28+ unit-тестов
